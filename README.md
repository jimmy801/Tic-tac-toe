# Tic-tac-toe

Instroduction
---

Tic-tac-toe，也就是井字遊戲，是使用`C#`寫的一個滑鼠點擊視窗介面小遊戲，並透過各種策略因此和電腦遊玩時，電腦方將立於不敗之地

Envorionment
---

-Windows

遊戲方法
---

1.選擇先下後下或讓電腦隨機選擇先下後下

2.選擇自己要O、X或隨機選擇
